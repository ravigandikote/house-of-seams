# House of Seams

Welcome to the **House of Seams** project! This is a contemporary boutique brand focused on lifestyle and jewelry, offering a range of custom and designer clothing, bridal couture, and curated jewelry pieces.

## Project Overview

This project is built using React and Next.js, leveraging modern web technologies to create a responsive and user-friendly experience. The website features a catalog of products, an admin dashboard for managing content, and various interactive components to enhance user engagement.

## Features

- **Home Page**: Showcases the brand's hero section, featured products, and categories.
- **About Page**: Details the brand story and philosophy.
- **Collections**: Displays various product collections.
- **Product Pages**: Lists all products with detailed views for individual items.
- **Gallery**: A visual showcase of the brand's offerings.
- **Testimonials**: Displays customer feedback and experiences.
- **Blog**: A section for articles and updates.
- **Contact Page**: Includes a contact form and Google Maps integration.
- **Booking System**: Allows users to book appointments.
- **FAQs**: Answers to common questions.
- **Policies**: Information on brand policies.
- **E-commerce Functionality**: Includes cart, wishlist, and checkout processes.
- **Admin Dashboard**: For managing products, categories, pricing, and bookings.

## Tech Stack

- **Frontend**: React, Next.js
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Data Fetching**: React Query
- **Forms**: React Hook Form
- **Payment Integration**: Razorpay or Stripe
- **Backend**: Supabase or Firebase for admin dashboard
- **Image Optimization**: Cloudinary or Next/Image

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/house-of-seams.git
   cd house-of-seams
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.example` to `.env` and fill in the required values.

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## Deployment

To deploy the application, you can use Vercel for Next.js applications:

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and sign in.
3. Import your GitHub repository.
4. Follow the prompts to deploy your application.
